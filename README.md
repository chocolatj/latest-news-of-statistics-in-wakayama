# latest-news-of-statistics-in-wakayama
　毎月、話題性、速報性のあるデータをグラフや表にしたものに加え、鉱工業生産指数、景気動向指数、消費者物価指数など、県内の最新の経済情報を提供しています。http://www.pref.wakayama.lg.jp/prefg/020300/news/index.html
